This program should parse TXT file and write it down in XML file, then it should load from that XML file and display in TreeView

*Example*:

```txt
1+2_1+Брусов Анатолий+25+10000+1 &rarr;    1   2_1      Брусов Анатолий 25  10000   1
                                            ID  Philial  Full Name       Age Salary  Internship 
```
